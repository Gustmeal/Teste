Sobre o Sistema de Deliberação de Pagamento

Este módulo permite criar relatórios automatizados de deliberação de pagamento para análise da GEADI. Os dados são extraídos automaticamente das tabelas do sistema e organizados em formato padronizado.

Funcionalidades
  • Busca automática de dados do contrato/imóvel
  • Preenchimento automático de dados do estoque
  • Identificação automática de períodos prescritos
  • Busca de processos judiciais e débitos
  • Salvamento e edição de deliberações
Fontes de Dados
  • MOV_TB012_IMOVEIS_NAO_USO_ESTOQUE
  • MOV_TB032_SISCALCULO_PRESCRICOES
  • MOV_TB031_SISCALCULO_CALCULOS
  • SISJUD - tblProcessoCredito e tblProcessoEmgea
  • MOV_TB017_DESPESAS_MANUTENCAO (SISDEX)
  • MOV_TB004_DESPESAS_ANALITICO (SISGEA)
