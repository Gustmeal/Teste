Sobre o Sistema de Deliberação de Pagamento

Este módulo permite criar relatórios automatizados de deliberação de pagamento para análise da GEADI. Os dados são extraídos automaticamente das tabelas do sistema e organizados em formato padronizado.

  • Busca automática: Dados de contratos, estoque, vendas e débitos
  • Cálculos precisos: Correção monetária com diversos índices econômicos
  • Exportação PDF: Relatórios prontos para impressão e assinatura
  • Controle de status: Acompanhamento do fluxo de aprovação
